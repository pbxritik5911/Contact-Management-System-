# 📱 Contact Management System (Flask)

A modern and user-friendly **Contact Management System** developed using **Flask (Python)** with a premium UI/UX design. This project allows users to efficiently manage contacts with full CRUD functionality and a clean, responsive interface.

---

## 🚀 Features

- ➕ Add new contacts (Name, Phone, Email)
- 📋 View all contacts in a modern card layout
- ✏️ Edit existing contact details
- ❌ Delete contacts instantly
- 🔍 Search contacts by name or phone number
- 🎨 Premium UI (Glassmorphism + Gradient Design)
- 📱 Fully responsive (Mobile + Desktop)

---

## 🛠️ Tech Stack

- **Backend:** Python, Flask
- **Frontend:** HTML, CSS, Bootstrap 5
- **Templating:** Jinja2
- **Storage:** In-memory (No database)
